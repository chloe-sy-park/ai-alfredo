# 🏗️ 알프레도 Information Architecture (IA)

> **버전**: v1.0  
> **작성일**: 2025-01-11  
> **목표**: ADHD 친화적 AI 버틀러의 정보 구조 설계

---

## 📊 IA 개요

### 핵심 원칙

| 원칙 | 설명 | ADHD 이유 |
|------|------|----------|
| **통합 대시보드** | 홈에서 모든 것을 한눈에 | 페이지 이동 = 맥락 상실 위험 |
| **3-depth 제한** | 최대 3단계까지만 | 깊은 네비게이션 = 이탈 |
| **컨텍스트 유지** | 어디서든 현재 상태 보임 | "내가 뭐 하고 있었지?" 방지 |
| **즉시 접근** | 핵심 기능 1-tap | 여러 단계 = 포기 |

### 페이지 역할 분리

```
┌─────────────────────────────────────────────────────────────────┐
│                         알프레도 앱                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   🏠 Home          💼 Work           🌿 Life          💬 Chat   │
│   ──────────       ──────────        ──────────       ──────── │
│   통합 대시보드     업무 집중 뷰       개인 케어 뷰      AI 대화   │
│   (Work + Life)    (업무만)          (일상만)         (알프레도) │
│                                                                 │
│   "오늘 하루        "일에만            "나를 위한        "도움이   │
│    전체 보기"       집중할 때"         시간"            필요할 때" │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 🔄 주요 User Flow

### Flow 1: 아침 루틴

```
알림 → Home → 브리핑 확인 → 탑3 선택 → 첫 태스크 시작
        │
        ↓
   컨디션 퀵체인지 (선택)
```

### Flow 2: 태스크 완료

```
Home/Work → 태스크 탭 → 완료 체크 → 축하 애니메이션 → XP 획득
                                          │
                                          ↓
                                    펭귄 반응 (선택)
```

### Flow 3: 집중 모드

```
Home "지금 집중" → 집중 시작 → 타이머 실행 → 완료/중단 → 기록
      또는
Work → 집중 모드 선택
      또는
Chat → 알프레도 제안 수락
```

### Flow 4: 브레인덤프

```
Chat → "브레인덤프" 선택 → 자유 입력 → 알프레도 정리 → 태스크 생성
                                              │
                                              ↓
                                        Home 탑3에 반영
```

### Flow 5: 인사이트 확인

```
Home 브리핑 "자세히" → Insights → 탭 선택 → 상세 분석
       또는
Settings → Insights 직접 접근
```

## 🔑 핵심 설계 결정

### 1. Home = 모든 것의 허브
- 페이지 이동 없이 대부분의 정보 확인 가능
- Work/Life는 **필터링된 뷰**일 뿐

### 2. Chat = 중앙 배치
- 알프레도와의 대화가 핵심 경험
- 네비게이션 중앙에서 강조

### 3. 3-tap 규칙
- 어떤 기능이든 3번 터치 내에 도달
- 깊은 depth 지양

### 4. 컨텍스트 유지
- 모든 페이지에서 현재 상태 표시 (펭귄, 탑3 등)
- 페이지 이동해도 맥락 유지

### 5. Progressive Disclosure
- 핵심 정보만 먼저
- 상세는 "자세히 보기"로
