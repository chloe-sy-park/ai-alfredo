## 🎯 핵심 설계 원칙

### ADHD 친화적 원칙
| 원칙 | 적용 방법 |
|------|----------|
| **낮은 인지 부하** | 선택지 최대 3개, 2-tap 인터랙션 |
| **즉각적 가치** | 3초 안에 "이 앱이 뭔지" 경험 |
| **용서하는 UX** | 스트릭 실패해도 괜찮음, 롤오버 |
| **외부 구조 제공** | 알프레도가 구조를 제안 |
| **도파민 보상** | 작은 성취도 축하, 펭귄 반응 |

### Permission Priming 원칙
| 순서 | 설명 |
|------|------|
| 1. 가치 먼저 | 권한 요청 전에 가치 경험 |
| 2. 맥락 설명 | "왜 필요한지" 한 줄로 |
| 3. 즉시 결과 | 연동 직후 "이게 당신이 얻는 것" |
| 4. 탈출 가능 | 언제든 스킵 가능 |

## 1️⃣ Phase 1: 발견 (Discovery)

### 사용자 상황
- 앱스토어/추천으로 알프레도 발견
- "또 다른 생산성 앱인가?" 의심
- ADHD 관련 키워드에 끌림

### 터치포인트
| 채널 | 메시지 | 목표 |
|------|--------|------|
| **앱스토어** | "ADHD 친화적 AI 버틀러 🐧" | 타겟 명확화 |
| **스크린샷** | 펭귄 캐릭터 + 심플 UI | 호기심 유발 |
| **설명** | "물어보지 않고 알아서" | 차별점 강조 |
| **리뷰** | "드디어 나를 이해하는 앱" | 사회적 증거 |

### 감정 곡선
```
호기심 😐 → 기대감 🤔
         "이건 좀 다를지도?"
```

### 알프레도 첫 메시지

```
🐧 알프레도:

"안녕하세요! 알프레도예요.
압도당하지 않게 하루를 정리해드리는 AI 버틀러입니다.

시작하기 전에—지금 가장 도움이 될 건 뭘까요?

[ 🗂️ 뭔가 정리하기 ]
[ 💭 머릿속에 있는 거 꺼내기 ]  
[ 🌿 그냥 부드러운 한마디 ]"
```

### 설계 포인트
| 요소 | 의도 |
|------|------|
| **3개 선택지** | ADHD 결정 피로 방지 |
| **Pi 스타일 질문** | 사용자에게 주도권 부여 |
| **가입 없이 시작** | 진입 장벽 최소화 |
| **버틀러 성격 확립** | 첫 인상으로 톤 설정 |

### 감정 곡선
```
기대감 🤔 → 놀람 😮 → 안도감 🙂
              "물어보는 게 아니네?"
              "내가 선택할 수 있네"
```

## 3️⃣ Phase 3: 온보딩 (Onboarding)

### 타이밍: Day 0 (5-10분)

### 목표
> **설문처럼 느껴지지 않게 정보 수집**  
> **가입/권한 요청 전에 가치 경험**

### 온보딩 플로우

```
┌─────────────────────────────────────────────────────────────────┐
│                        온보딩 플로우                              │
│                                                                  │
│   Step 1          Step 2          Step 3          Step 4        │
│  ┌──────┐        ┌──────┐        ┌──────┐        ┌──────┐       │
│  │ 첫   │   →    │ 가치 │   →    │ 권한 │   →    │ 즉시 │       │
│  │ 대화 │        │ 경험 │        │ 요청 │        │ 결과 │       │
│  └──────┘        └──────┘        └──────┘        └──────┘       │
│   "뭐가           Quick           Google          "여기         │
│   필요해요?"      Win 제공        연동 제안        당신이        │
│                                                   얻는 것"       │
│                                                                  │
│  ─────────────────────────────────────────────────────────────  │
│  가입 없이        탑3 추천        맥락 설명        캘린더 미리   │
│  대화 시작        체험            투명하게         보기 제공     │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### Step 1: 첫 대화 (대화형 정보 수집)

```
🐧 알프레도:

(사용자가 "뭔가 정리하기" 선택 시)

"좋아요! 지금 머릿속에 있는 거, 
 일단 다 저한테 쏟아내 보세요.

 정리는 제가 할게요 ✨"

────────────────────────────
[사용자 입력]
"내일까지 보고서 써야 하고, 
 치과 예약도 해야 하고,
 팀 미팅 준비도 해야 하는데
 뭐부터 해야 할지 모르겠어요"
────────────────────────────

🐧 알프레도:

"와, 머리가 복잡했겠어요.
 제가 정리해봤어요:

 📋 오늘 할 일
 ├─ 🔴 보고서 작성 (내일 마감)
 ├─ 🟡 팀 미팅 준비
 └─ 🟢 치과 예약

 이 중에서 오늘 가장 중요한 건 뭘까요?
 
 [ 보고서 ] [ 미팅 준비 ] [ 치과 ]"
```

### Step 2: Quick Win (즉각적 가치 경험)

```
🐧 알프레도:

"보고서가 제일 급하시군요!

 💡 제안: 보고서를 작은 단계로 나눠볼까요?
 
 [ 네, 좋아요 ] [ 일단 넘어갈게요 ]"

────────────────────────────
(사용자가 "네, 좋아요" 선택)
────────────────────────────

🐧 알프레도:

"보고서 작성" 이렇게 나눠봤어요:

 □ 1. 자료 모으기 (15분)
 □ 2. 개요 잡기 (10분)  
 □ 3. 초안 쓰기 (30분)
 □ 4. 다듬기 (15분)

 작은 조각으로 나누니까 
 좀 덜 무섭지 않아요? 🐧

 ✨ 이제 첫 번째 "자료 모으기"만 
    집중하면 돼요!
```

**🎯 Magic Moment**: 막연한 "보고서"가 구체적 단계로 분해되는 순간

### Step 3: 권한 요청 (Permission Priming)

```
🐧 알프레도:

"잠깐, 한 가지 물어볼게요.

 캘린더를 연결해주시면 
 제가 더 잘 도와드릴 수 있어요:

 ✓ 미팅 시간 알아서 피해서 집중 시간 추천
 ✓ 일정 30분 전에 자연스럽게 리마인드
 ✓ 바쁜 날은 톤 더 부드럽게 조절

 📊 캘린더는 패턴 분석에만 쓰이고,
    내용은 저장하지 않아요.

 [ 🔗 Google 캘린더 연결 ]
 [ ⏭️ 나중에 할게요 ]"
```

### Step 4: 즉시 결과 시연

```
🐧 알프레도:

"연결 완료! 🎉

 잠깐 살펴봤는데요...

 📅 내일 일정
 ├─ 10:00 팀 미팅 (1시간)
 ├─ 14:00 치과 예약 되어있네요!
 └─ 오후는 비어있어요

 💡 발견한 것:
 "내일 오후가 보고서 쓰기 좋겠어요.
  오전 미팅 끝나고 점심 먹고 나면
  2시간 정도 집중할 수 있을 것 같아요."

 ✨ 이렇게 매일 아침 알려드릴게요!"
```

### 온보딩 종료

```
🐧 알프레도:

"오늘 저랑 얘기해주셔서 고마워요!

 마지막으로, 편하게 불러주세요.
 이름이 어떻게 되세요?

 ────────────────────────────
 [이름 입력] 또는 [ 그냥 넘어갈게요 ]
 ────────────────────────────

 (이름 입력 시)
 
 "[이름]님, 반가워요! 🐧
  
  내일 아침에 브리핑으로 찾아올게요.
  오늘 하루 마무리 잘 하세요! 💜"
```

### 감정 곡선
```
안도감 🙂 → 성취감 😊 → 기대감 🤩
        "이거 진짜 도움 되네"
        "내일 아침이 기대돼"
```
